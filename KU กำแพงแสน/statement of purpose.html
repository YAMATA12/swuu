statement of purpose

           สวัสดีครับคณะกรรมการทุกท่าน ผมชื่อ นายธนกร ประดับเวช หรือเรียกสั้น ๆ ว่า "กร" ผมมีความสนใจในด้านเทคโนโลยีและคอมพิวเตอร์มาตั้งแต่เด็ก ๆ การเรียนรู้เกี่ยวกับการเขียนโปรแกรมและพัฒนาเว็บไซต์เป็นสิ่งที่ผมหลงใหลและสนุกกับมันอย่างมาก ตั้งแต่ช่วงที่ได้ลองเขียนโค้ด HTML และ CSS ครั้งแรก ผมรู้สึกว่าเทคโนโลยีสามารถสร้างสรรค์สิ่งใหม่ ๆ และช่วยแก้ปัญหาได้จริง ทำให้ผมอยากเรียนรู้และพัฒนาทักษะในสาขาวิศวกรรมคอมพิวเตอร์มากยิ่งขึ้น
           การที่ผมได้ทดลองทำโปรเจกต์เล็ก ๆ ด้วย HTML และ CSS ตั้งแต่สมัยเรียนมัธยมปลาย ถือเป็นจุดเริ่มต้นที่ทำให้ผมสนใจในการเรียนรู้เกี่ยวกับการพัฒนาเว็บไซต์และเทคโนโลยีอื่น ๆ เช่น JavaScript และการออกแบบเว็บไซต์ให้ตอบสนองการใช้งานที่ดี เมื่อมีโอกาสฝึกทำเว็บไซต์ส่วนตัว ผมจึงได้พัฒนาทักษะในการเขียนโค้ดและเรียนรู้การใช้งานเครื่องมือต่าง ๆ ที่เกี่ยวข้อง นอกจากนั้นยังได้ทดลองทำโปรเจกต์ที่เกี่ยวข้องกับการจัดการข้อมูลและการพัฒนาแอปพลิเคชันเบื้องต้น ซึ่งช่วยเสริมสร้างความมั่นใจและทักษะในการแก้ปัญหาทางเทคนิค การได้ทดลองทำงานร่วมกับทีมในการพัฒนาโปรเจกต์ต่าง ๆ ยังช่วยให้ผมได้เรียนรู้ถึงความสำคัญของการทำงานเป็นทีม การสื่อสารและการร่วมมือกันในการแก้ไขปัญหาต่าง ๆ อย่างมีประสิทธิภาพ ทั้งหมดนี้ได้เพิ่มพูนความรู้และทักษะที่จำเป็นสำหรับการพัฒนาซอฟต์แวร์ในอนาคต
          ผมเลือกศึกษาต่อที่คณะวิศวกรรมศาสตร์ สาขาวิศวกรรมคอมพิวเตอร์ของมหาวิทยาลัยศรีนครินทรวิโรฒ (มศว) เพราะเชื่อมั่นในคุณภาพการศึกษาที่มีมาตรฐานสูง และหลักสูตรที่มีความครอบคลุมทั้งในด้านทฤษฎีและการปฏิบัติที่สามารถเตรียมความพร้อมให้กับนักศึกษาสำหรับการทำงานในสาขาวิศวกรรมคอมพิวเตอร์ การที่มหาวิทยาลัยมีคณาจารย์ผู้ทรงคุณวุฒิ และการสนับสนุนด้านการเรียนการสอนที่ทันสมัย ทำให้มั่นใจได้ว่าจะได้รับความรู้ที่สามารถนำไปประยุกต์ใช้ในการทำงานได้จริง
          ผมมองว่าการเรียนในคณะวิศวกรรมศาสตร์ สาขาวิศวกรรมคอมพิวเตอร์ที่ มศว จะทำให้ผมได้พัฒนาทักษะทั้งในด้านการเขียนโปรแกรม การออกแบบระบบคอมพิวเตอร์ และการแก้ปัญหาทางเทคนิคที่ซับซ้อน โดยเฉพาะการได้เรียนรู้จากประสบการณ์จริงในห้องปฏิบัติการและการทำโปรเจกต์ร่วมกับเพื่อน ๆ จะช่วยเพิ่มพูนความรู้และความสามารถที่สามารถนำไปใช้ในการพัฒนาเทคโนโลยีในอนาคตได้
          ผมตั้งใจที่จะพัฒนาความรู้ด้านวิศวกรรมคอมพิวเตอร์และเทคโนโลยีให้ลึกซึ้งยิ่งขึ้น โดยมีเป้าหมายที่จะเป็นผู้พัฒนาซอฟต์แวร์ที่สามารถสร้างสรรค์และพัฒนานวัตกรรมใหม่ ๆ ที่มีประโยชน์ต่อสังคมและธุรกิจ ในระยะยาว ผมอยากทำงานในด้านวิศวกรรมซอฟต์แวร์ หรือการพัฒนาแอปพลิเคชัน ที่มีบทบาทสำคัญในการสร้างระบบที่สามารถช่วยแก้ปัญหาต่าง ๆ ได้อย่างมีประสิทธิภาพ และพร้อมที่จะเผชิญกับความท้าทายในโลกดิจิทัลที่มีการเปลี่ยนแปลงอย่างรวดเร็ว การเรียนต่อที่คณะวิศวกรรมศาสตร์ สาขาวิศวกรรมคอมพิวเตอร์ที่ มศว เป็นโอกาสสำคัญที่จะช่วยให้ผมสามารถพัฒนาทักษะและความรู้ในด้านที่ผมหลงใหล และเสริมสร้างความสามารถในการทำงานในอุตสาหกรรมเทคโนโลยี ผมมั่นใจว่า มศว จะเป็นสถานที่ที่เหมาะสมในการเรียนรู้และเติบโตในเส้นทางที่ผมตั้งเป้าหมายไว้ ขอบพระคุณคณะกรรมการการรับสมัครที่ให้โอกาสในการพิจารณาใบสมัครของผม หวังว่าจะได้เป็นส่วนหนึ่งของมหาวิทยาลัยศรีนครินทรวิโรฒในอนาคต
